Qu'est-ce qu'un IDE ? 🖥️

Un IDE (Environnement de Développement Intégré) est un logiciel qui facilite l'écriture et l'exécution de code. Il propose des fonctionnalités comme la coloration syntaxique, l'auto-complétion et l'affichage des erreurs.

Voici quelques IDE populaires :

  • Replit : Un IDE en ligne, simple et rapide. Utiliser Replit comme IDE
  • Jupyter Notebook : Idéal pour les débutants et l'expérimentation. Utiliser Jupyter Notebook comme IDE
  • VS Code : Un IDE puissant avec de nombreuses extensions. Utiliser VS code comme IDE

Choisis l'IDE qui te convient pour exécuter ton code Python. Clique ici pour en savoir plus

Qu'est-ce qu'une bibliothèque en Python ? 📚

Une bibliothèque est un ensemble de fonctions et de modules prêts à l'emploi, qui permettent d'effectuer des tâches spécifiques sans avoir à les coder soi-même.

Par exemple, la bibliothèque turtle permet de dessiner facilement avec Python. Plutôt que de programmer nous-mêmes comment déplacer un crayon sur l'écran, nous pouvons utiliser les fonctions déjà fournies par turtle.

Les commentaires en Python 📝

En Python, nous pouvons écrire des commentaires en utilisant le symbole #. Tout ce qui est écrit après # dans une ligne ne sera pas exécuté par Python. Cela permet d'ajouter des explications pour rendre le code plus compréhensible.

# Ceci est un commentaire, Python ne l'exécutera pas

# Il est utile pour expliquer le code

import turtle # Importe la bibliothèque Turtle

Qu'est-ce que Turtle ?

Le module turtle est une bibliothèque Python qui permet de dessiner à l'écran en déplaçant une "tortue virtuelle".

Commençons par importer le module :

import turtle

Déplacer la tortue

Voici quelques commandes de base :

  • turle.Turtle() : Créer une "tortue"
  • turtle.forward(100) : Avancer de 100 pixels
  • turtle.right(90) : Tourner à droite de 90°
  • turtle.left(90) : Tourner à gauche de 90°
  • turtle.done() : Fermer la fenêtre en cliquant

import turtle #Importation de la bibliothèque Turtle
t = turtle.Turtle() # Créer une tortue
t.forward(100) # Avance de 100 pixels
t.right(90) # Tourne à droite de 90 degrés
t.forward(100) # Avance à nouveau
t.done() # Fenêtre refermée en cliquant

Personnaliser la tortue

Nous pouvons changer son apparence :

  • turtle.color("blue") : Change la couleur
  • turtle.speed(3) : Définit la vitesse
  • turtle.shape("turtle") : Donne la forme d’une tortue

Pour voir toutes les fonctionnalités du module turtle
import turtle #Importation de la bibliothèque Turtle
turtle.color("blue") #Couleur bleue
turtle.speed(3) # Vitesse modifiee
turtle.shape("turtle") # Forme Tortue

Les bases de Python 🐍

Affichage avec print() 🖨️

La fonction print() permet d'afficher du texte ou des valeurs dans la console.

Exemple :

# Afficher du texte
print("Bonjour, bienvenue dans Python !")

# Afficher un nombre
print(42)

# Afficher plusieurs éléments
print("Le résultat est :", 10 + 5)

Variables et types de données 🔢

Une variable est une boîte qui stocke une valeur. On lui attribue un nom et une valeur.

Python propose plusieurs types de données :

  • int : nombres entiers (10, -3)
  • float : nombres décimaux (3.14, 2.5)
  • str : chaînes de caractères ("Hello")
  • bool : valeurs booléennes (True, False)

# Déclaration de variables
nom = "Alice" # Chaîne de caractères (str)
age = 25 # Nombre entier (int)
taille = 1.70 # Nombre décimal (float)
est_etudiant = True # Valeur booléenne (bool)
# Affichage des valeurs
print("Nom :", nom)
print("Âge :", age)
print("Taille :", taille, "m")
print("Étudiant :", est_etudiant)

Entrée utilisateur avec input() ⌨️

La fonction input() permet de demander à l'utilisateur de saisir une valeur. Cette valeur est toujours sous forme de texte (str).

# Demander le prénom de l'utilisateur
prenom = input("Quel est ton prénom ? ")
# Afficher un message personnalisé
print("Bonjour", prenom, "!")

⚠️ Si l'on veut convertir une saisie en nombre, on utilise int() ou float().

# Demander un âge et le convertir en entier
age = int(input("Quel est ton âge ? "))
print("Dans 5 ans, tu auras", age + 5, "ans !")

Manipulation des chaînes de caractères ✍️

Python propose plusieurs opérations sur les chaînes de caractères :

  • Concaténation : "Bonjour" + " " + "Alice" → "Bonjour Alice"
  • Longueur : len("Python") → 6
  • Majuscules/minuscules : "python".upper(), "PYTHON".lower()
  • Remplacement : "Salut Bob".replace("Bob", "Alice") → "Salut Alice"

texte = "Bonjour Python"
# Longueur du texte
print("Longueur :", len(texte))
# Mettre en majuscules
print("Majuscules :", texte.upper())
# Remplacer un mot
print("Modification :", texte.replace("Python", "le monde"))

Conditions et prise de décision ⚖️

Les conditions en Python 🧐

Une condition permet d’exécuter un bloc de code seulement si une certaine situation est remplie. En Python, on utilise if, elif et else.

Voici un exemple simple :

age = 18
if age >= 18:
    print("Tu es majeur.")
else:
    print("Tu es mineur.")

Les comparateurs 🔍

Python utilise des opérateurs de comparaison pour évaluer des conditions :

  • == : égal à
  • != : différent de
  • < : inférieur à
  • > : supérieur à
  • <= : inférieur ou égal à
  • >= : supérieur ou égal à

Exemple :

note = 15
if note == 20:
    print("Parfait !")
elif note >= 10:
    print("Tu as la moyenne.")
else:
    print("Tu dois progresser.")

Les boucles 🔁

La boucle while 🔄

Une boucle while permet d’exécuter un bloc de code tant qu’une condition est vraie.

Exemple :

compteur = 0
while compteur < 5:
    print("Tour :", compteur)
    compteur += 1 # On ajoute 1 à chaque tour

La boucle for ➰

Une boucle for est utilisée pour parcourir une séquence (liste, chaîne de caractères, etc.).

Exemple :

for lettre in "Python":
    print(lettre)

Utilisation de range() 📏

La fonction range() permet de générer une suite de nombres. Elle est souvent utilisée avec une boucle for.

Exemples :

# Compter de 0 à 4
for i in range(5):
    print(i)
# Compter de 1 à 10 avec un pas de 2
for i in range(1, 11, 2):
    print(i)

Les fonctions 🛠️

Qu'est-ce qu'une fonction ? 🤔

Une fonction est un bloc de code réutilisable qui exécute une tâche spécifique. On définit une fonction avec le mot-clé def.

Exemple simple :

def dire_bonjour():
    print("Bonjour !")
# Appel de la fonction
dire_bonjour()

Les paramètres et les retours de fonctions 🎯

Une fonction peut recevoir des paramètres et retourner une valeur avec return.

Exemple :

def addition(a, b):
    return a + b
resultat = addition(5, 3)
print("Résultat :", resultat)

Listes et tuples 📋

Les listes 📜

Une liste est une structure qui permet de stocker plusieurs valeurs dans une seule variable. Les éléments d'une liste sont modifiables et indexés.

Exemple :

# Création d'une liste
fruits = ["pomme", "banane", "cerise"]
# Accéder à un élément
print(fruits[0]) # Affiche "pomme"
# Modifier un élément
fruits[1] = "orange"
# Ajouter un élément
fruits.append("kiwi")
# Supprimer un élément
fruits.remove("cerise")
print(fruits)

Les tuples 🔗

Un tuple est similaire à une liste, mais ses éléments sont immuables (on ne peut pas les modifier après leur création).

Exemple :

# Création d'un tuple
coordonnees = (10, 20)
# Accéder aux éléments
print(coordonnees[0]) # Affiche 10
# Les tuples ne peuvent pas être modifiés
# coordonnees[0] = 15 # ⚠️ Erreur ! Un tuple est immuable

Les dictionnaires et manipulation de données 📚

Les dictionnaires 📖

Un dictionnaire est une structure de données qui stocke des informations sous forme de clé → valeur. Contrairement aux listes, les dictionnaires permettent d'accéder rapidement aux données via une clé.

Exemple :

# Création d'un dictionnaire
joueur = {
"nom": "Alex",
"score": 150,
"niveau": 5
}
# Accéder à une valeur
print(joueur["nom"]) # Affiche "Alex"
# Modifier une valeur
joueur["score"] = 200
# Ajouter une nouvelle clé
joueur["vie"] = 3
print(joueur)

Parcourir un dictionnaire avec une boucle 🔄

On peut parcourir un dictionnaire pour afficher toutes ses clés et valeurs avec une boucle for.

Exemple :

# Parcourir un dictionnaire
for cle, valeur in joueur.items():
    print(cle, ":", valeur)

Projet 1 : Le Juste Prix 💰

Crée un jeu où l'utilisateur doit deviner un nombre aléatoire entre 1 et 100. À chaque tentative, le programme indique si le nombre est trop grand ou trop petit.

import random
# Génération d'un nombre aléatoire
nombre_secret = random.randint(1, 100)
tentative = 0

while True:
tentative += 1
guess = int(input("Devine le nombre : "))

if guess < nombre_secret:
print("C'est plus !")
elif guess > nombre_secret:
print("C'est moins !")
else:
print(f"Bravo ! Trouvé en {tentative} essais.")
break
